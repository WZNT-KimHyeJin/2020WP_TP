고양이와 춤을!

무작위로 주어지는 방향키 정보와 일치하게 방향키를 입력하는 게임이다.

관냥골 인기스타인 나나는 관냥골에서 벗어나 전국구 슈퍼스타가 되고 싶은 냥이다.

마침 관냥골에서 유명한 댄스 대회인 "고양이와 춤을!" 이라는 대회가 열리게 되고

나나는 망설임 없이 참여하게 된다.

근데 춤추는 걸 나보고 도와달라고...?

내가??? 어떻게...??

게임 방법

1. play 화면에 방향을 나타내는 그림이 랜덤으로 보여집니다.



→ : 방향키 오른쪽 ← : 방향키 왼쪽 ↑:방향키 위쪽
↓ : 방향키 아래쪽 ■ : 스페이스바


2. 해당 그림에 맞는 방향키를 누르면 고양이가 춤을 춥니다.

...

3. 한 stage를 성공하기 위해서는 무작위로 배치되는 세가지의 키들을
주어지는 세번의 기회 안에 올바르게 입력하면 성공입니다.


4. stage 별로 입력해야 하는 key의 개수는 많아지고,
stage를 성공하기 위해 주어지는 시간은 짧아집니다.


5. 보너스 단계는 입력해야 하는 키값이 기존의 방식처럼
키보드의 방향키와 스페이스바가 아닌 랜덤으로 정해집니다.
주어지는 키값에 해당하는 키보드를 정확하게 누르면 보너스 단계를 CLEAR!
